# Privacy Policy — Driftmoon Sleep Uploader

_Last updated: July 7, 2026_

Driftmoon Sleep Uploader is a personal, single-user automation tool that
generates original sleep soundscape videos and uploads them to its owner's
own YouTube channel (https://www.youtube.com/@DriftMoonSleep) via the
YouTube Data API v3.

## Data we access
The tool uses OAuth 2.0 to access **only the channel owner's own YouTube
account**, with the sole purpose of uploading videos and setting their
thumbnails and metadata. It does not access, collect, store, or process any
data belonging to other YouTube users or third parties.

## Data we store
The only credential stored is the channel owner's OAuth refresh token, kept
encrypted as a GitHub Actions secret. No YouTube API data is stored,
shared, sold, or used for any other purpose. The tool has no database and
no users other than its owner.

## Third parties
No data is shared with any third party. The tool runs on GitHub Actions
infrastructure solely to execute the upload process.

## Google user data
Use of information received from Google APIs adheres to the
[Google API Services User Data Policy](https://developers.google.com/terms/api-services-user-data-policy),
including the Limited Use requirements, and to the
[YouTube API Services Terms of Service](https://developers.google.com/youtube/terms/api-services-terms-of-service).
Users can revoke access at any time via
[Google security settings](https://myaccount.google.com/permissions).
